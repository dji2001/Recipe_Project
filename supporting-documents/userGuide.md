## Instructions

Upon running, the program will welcome the user! Then, it will prompt the user to press "Enter" to continue.
Next, the program will ask the user for four options: 
 1. View All Recipes
 2. Search By Name
 3. Create Recipe
 4. To Exit: Type: Exit
